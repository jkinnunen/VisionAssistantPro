Vision Assistant Pro

Vision Assistant Pro is an advanced, multi-modal AI assistant for NVDA. It leverages Google's Gemini models to provide intelligent screen reading, translation, voice dictation, and document analysis capabilities.

This add-on was released to the community in honor of the International Day of Persons with Disabilities.

1. Setup & Configuration

Go to NVDA Menu > Preferences > Settings > Vision Assistant Pro.

  • API Key: Required. Get a free key from Google AI Studio.
  • Model: Choose gemini-2.5-flash-lite (Fastest) or standard Flash models.
  • Languages: Set Source, Target, and AI Response languages.
  • Smart Swap: Automatically swaps languages if the source text matches the target language.

2. Global Shortcuts

To ensure maximum compatibility with laptop layouts, all shortcuts use NVDA + Control + Shift.

NVDA + Ctrl + Shift + T : Smart Translator
Translates the text under the navigator cursor. If text is selected, it prioritizes the selection.
NVDA + Ctrl + Shift + Y : Clipboard Translator
Translates the content currently in the clipboard. Recommended for Web Browsers where NVDA selection might be inconsistent.
NVDA + Ctrl + Shift + S : Smart Dictation
Converts your speech to text.
  • Press Once: Start recording (High Beep). Speak naturally.
  • Press Again: Stop recording (Low Beep). The AI transcribes, fixes grammar, and types the text for you.
NVDA + Ctrl + Shift + R : Text Refiner
Opens a menu to Summarize, Fix Grammar, Explain, or run Custom Prompts on the selected text.
NVDA + Ctrl + Shift + C : CAPTCHA Solver
Captures the CAPTCHA image under the navigator object, solves it, and automatically types the code.
NVDA + Ctrl + Shift + V : Object Vision
Takes a screenshot of the navigator object and describes it. Opens a chat window for follow-up questions.
NVDA + Ctrl + Shift + O : Full Screen Vision
Scans the entire screen. You can ask the AI to describe the overall layout, read visible text, or analyze the UI.
NVDA + Ctrl + Shift + D : Document Analysis
Select a file (PDF, TIFF, TXT, MD, PY) to chat with. You can ask questions about its content.
NVDA + Ctrl + Shift + A : Audio Transcription
Select an audio file (MP3, WAV, OGG) to transcribe it to text.
NVDA + Ctrl + Shift + L : Last Translation
Re-reads the last translated text without using the API.
NVDA + Ctrl + Shift + U : Update Check
Manually checks GitHub for the latest version of the add-on.

3. Custom Prompts & Variables

You can create custom commands in the Settings dialog using the format: Name:Prompt Text.
Separate multiple commands with a pipe symbol | or new lines.

Available Variables

Variable Description Input Type
[selection] Inserts the currently selected text. Text
[clipboard] Inserts the clipboard content. Text
[screen_obj] Attaches a screenshot of the Navigator Object. Image
[screen_full] Attaches a screenshot of the Full Screen. Image
[file_ocr] Opens a dialog to select a file. The AI will extract text or analyze visual content.
Defaults to "Extract text" if no prompt is given.		 Image, PDF, TIFF
[file_read] Opens a dialog to select a text document. TXT, Code, PDF
[file_audio] Opens a dialog to select an audio file. MP3, WAV, OGG

Example Custom Prompts

  • Quick OCR:
    My OCR:[file_ocr]
    (Opens a file picker, sends the image/PDF, and asks AI to extract text.)
  • Translate Image:
    Translate Img:Extract text from this image and translate to Persian. [file_ocr]
  • Analyze Audio:
    Summarize Audio:Listen to this recording and summarize the main points. [file_audio]
  • Code Debugger:
    Debug:Find bugs in this code and explain them: [selection]
Note: File uploads are limited to 15MB. An active internet connection is required for all AI features. Multi-page TIFFs are automatically processed.
